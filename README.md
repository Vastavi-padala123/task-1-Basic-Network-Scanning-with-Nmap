# Task 1: Basic Network Scanning with Nmap

## Objective

To perform basic network scanning using Nmap and identify open ports, services, and host information.

##  Tools Used

* Nmap (Network Mapper)

##  Description

Nmap is a powerful open-source tool used for network discovery and security auditing by scanning systems to detect open ports and running services.

##  Steps Performed

1. Installed Nmap on the system.
2. Selected a target IP address or hostname.
3. Performed different types of scans:

   * Basic scan
   * SYN scan
   * Service version detection
4. Analyzed the scan results.

##  Sample Commands

nmap 10.10.103.14

nmap -sV 10.10.103.14

nmap -p- 10.10.103.14

Sample Output
Nmap provides details such as:

Open ports

Service names

Device type

Operating system (if detectable)


## ✅ Conclusion

Basic network scanning with Nmap helps in identifying vulnerabilities and understanding network structure, which is essential for security assessment.
